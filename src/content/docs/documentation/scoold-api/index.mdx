The Scoold REST API is accessible only if enabled with `scoold.api_enabled = true`. Authentication uses a `Bearer` access token (API key) generated on the Scoold Administration page.

<Aside type="tip">
  You can also browse the full interactive API console, available at `/apidocs` on your running Scoold instance.
</Aside>
