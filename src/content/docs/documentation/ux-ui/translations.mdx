You can translate Scoold to your language by copying the [English language file](https://github.com/Erudika/scoold/blob/master/src/main/resources/lang_en.properties)
and translating it, or click the "Edit file" on GitHub. 
<Image src={import(`../../../../images/documentation/translating.png`)}
  alt="Translating files on GitHub"
  class="py-5 border border-neutral-400 rounded"
  loading="eager"
/>

<Steps>
1. Copy the English file [↗ English language file](https://github.com/Erudika/scoold/blob/master/src/main/resources/lang_en.properties)
2. Rename file name from `lang_en.properties` to `lang_xy.properties` where `xy` is the language code for your locale. 
3. When the file is translated, [open a pull request on GitHub](https://github.com/Erudika/scoold/pulls) containing the new file
</Steps>

You can also change the default language of Scoold for all users by adding this to your Scoold configuration:
```ini title=scoold-application.conf
scoold.default_language_code = "en"
```
Replace `"en"` with the two-letter language code of your choice.
